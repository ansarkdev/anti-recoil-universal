A somewhat simplified anti-recoil script featuring dynamic gun configuration switching.

## Features

* **Simplified Logic:** Easy to read and modify for your own needs.
* **Gun Config Switching:** Seamlessly swap between different recoil profiles for different weapons on the fly.
